# Screenshots

Esta pasta contém os screenshots do aplicativo.

## Como adicionar screenshots:

1. **Tire prints** do app no celular
2. **Transfira** as imagens para esta pasta
3. **Renomeie** seguindo o padrão:
   - `01-tela-inicial.png`
   - `02-sugestoes.png`
   - `03-resultado-niteroi.png`
   - `04-resultado-sao-paulo.png`
   - `05-historico.png`
   - `06-erro.png`

4. **Atualize** o README.md substituindo os placeholders:

Substituir:
```markdown
<img src="https://via.placeholder.com/250x500/E8F4F8/2E88C7?text=Tela+Inicial" alt="Tela Inicial" width="200"/>
```

Por:
```markdown
<img src="./screenshots/01-tela-inicial.png" alt="Tela Inicial" width="200"/>
```

## Dicas para tirar bons screenshots:

- ✅ Use fundo limpo
- ✅ Tire em modo retrato
- ✅ Mostre diferentes estados do app
- ✅ Capture telas com dados reais
- ✅ Mostre as cores diferentes (sol, chuva, nublado)
